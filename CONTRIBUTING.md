# Contributing

Thanks for helping improve agent-sandbox!

## Development

```bash
npm install
npm run build
npm run self-validate
```

## Pull Requests

- Keep changes focused and minimal.
- Add or update validation scripts when behavior changes.
- Run `npm run self-validate` before submitting.

## Issues

Please include:
- OS + Docker version
- Exact command or snippet
- Logs/output (redact secrets)
